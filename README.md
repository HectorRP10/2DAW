# Librerías en Local vs CDN

## Librerías descargadas en local

### Ventajas

-   No necesita conexión a Internet.
-   Mayor control de la versión.
-   No depende de servicios externos.

### Desventajas

-   Ocupan espacio en el proyecto.
-   Hay que actualizarlas manualmente.
-   No aprovechan la caché de otros sitios web.

---------------------------------------------------------------------------

## Librerías usando CDN

### Ventajas

-   Carga rápida desde servidores externos.
-   No aumentan el tamaño del proyecto.
-   Fácil de implementar.

### Desventajas

-   Necesita conexión a Internet.
-   Depende de un servicio externo.
-   Puede fallar si el CDN no funciona.
